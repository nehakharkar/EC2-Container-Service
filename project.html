Steps to be Followed:

1. Create your docker file : An application ready to be deployed.

2. Go to Amazon ECS (Elastic Container Service) to get started, click on Repositories to create a new one.

3. Name the Repository as: ecs-assignment and click Next
This will create the Repository, save the Repository URI .



4. Now follow the steps given and run the commands to build, tag and push the docker image.









5. Go to Cluster part in the Amazon ECS Dashboard to create a new cluster. Select appropriate cluster template.
For this assignment we are going to select EC2 Linux + Networking template, click next.

6. Type cluster name as ECS-cluster-demo, select EC2 Instance type as t2.micro.
Select appropriate key pair. And Create an VPC which help in configuring the container instances to use.
Select the created VPC,subnets and security groups and click Create to create the cluster.







7. Go to ECS Dashboard to create a task. Task defines no of containers to be created. We would create a single container.
Click on task Definition and click on create new task Definition. Select EC2 and click on next button.
Name the task Definition as ECS-task-demo.



8. Click on Add Container to add a container instance.
Name the container as ECS-container-demo, enter the saved Repository URI from step in Image.
Enter memory limits as 300, host port as 80 and container port as 8000, cpu limits as 10 leaving all the other values as it is.



Click on Add button after entering all the values and click create.

9. Now we will create a service within the cluster. Go to Clusters in ECS Dashboard.
Select our created cluster and click create in the services tab.



10. Select EC2 as launch type, select appropriate task and cluster from the dropdown.
Name the service as ECS-service-demo and enter no of tasks as 1.



Click next three times and click on create service.



11. Thus our EC2 Instance is now launched. Connect it via SSH and save the screenshot as the linux-proof.png

12. Similarly create the windows instance and save the screenshot as windows-proof.png
